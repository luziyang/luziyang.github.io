mongodb入门1-数据库的分类

1 : Sql 数据库 : 支持Sql语言的数据库 . Oracl、MySql…
2 : NoSql 数据库 : 不支持Sql语言的数据库 . Redis , MongoDb…

两种数据库有n多区别..
Sql 数据库 NoSql数据库
实施一致性 简单快捷
事物 方便扩展
多表联合查询 更好的性能

MongDb :
1、无数据结构限制 : 没有表结构的限制，每条数据可以有完全不同的结构
业务开发方便快捷
sql数据库需要先定义表结构在使用
2、完全的索引支持 : redis的key-value
hbase的但索引，二级索引需要自己实现。
单键索引，多建索引: {x:1,y:1}
数组索引 : [“apple”,”lemon”]
全文索引 : “i am luziyang”
地理位置索引 : 2D
3、方便的冗余和扩展 : 复制集保证数据安全
分片扩展数据规模
4、良好的支持 : 完善的文档
齐全的驱动支持
